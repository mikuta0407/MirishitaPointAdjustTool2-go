mltd-pat: **M**illion **L**ive **T**heater **D**ays **P**oint **A**djust **T**ool

## 使い方(予定)
- `go install github.com/mikuta0407/mltd-pat@latest`
- 周年イベなら `mltd-pat aniv <現在ポイント> <目標ポイント>
- pointlist.tomlにライブ定義を書けます。
- ライブ総数を最小化したときの各ライブの必要回数が出力されます(予定)
